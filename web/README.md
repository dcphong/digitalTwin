# Smart Parking Web

Dashboard web cho Digital Twin Smart Parking.

## Stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- shadcn/ui
- Motion
- QRCode React

## Chạy local

```bash
npm install
npm run dev
```

Mở `http://localhost:3000`.

## Kiểm tra

```bash
npm run lint
npm run build
```
